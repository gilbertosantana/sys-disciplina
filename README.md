# Sobre o projeto
O sys-disciplina é uma aplicação backend desenvolvida no segundo roteiro de laboratório do minicurso de REST APIs, oferecido pelo Campus IV da UFPB.

A aplicação consiste em reunir todas as disciplinas cadastradas do curso de SI (Sistemas de Informação). Nela, os alunos poderão adicionar comentários a uma determinada disciplina, deixar seu like, atribuir uma nota e associar tags à disciplina. Também será possível visualizar todas as disciplinas cadastradas, pesquisar por uma disciplina específica, consultar médias, quantidades de likes e comentários, além de realizar diversos tipos de buscas.

# Tecnologias utilizadas
## Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven
  
# Como executar o projeto

## Back end
Pré-requisitos: Java 21

```bash
# clonar repositório

git clone https://github.com/gilbertosantana/sys-disciplina

# entrar na pasta do projeto

# executar o projeto
./mvnw spring-boot:run
```
